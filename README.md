# CustomWords

It allows the user to add custom words to its app.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'custom_words'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install custom_words

## Usage

```ruby
custom :my_word
```
